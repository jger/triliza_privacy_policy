Privacy Policy for Triliza

Last Updated: January 3, 2026
App Version: 1.15.2
Effective Date: January 1, 2025

This Privacy Policy describes how Triliza ("we," "our," or "us") collects, uses, and protects your information when you use our mobile application Triliza (the "App").

1. Information We Collect

1.1 Analytics Data (Optional)

With your explicit consent, we may collect anonymous analytics data to improve the App's performance and user experience:

  • Game Session Information: Game mode, board size, duration, moves count, and game results
  • App Usage Patterns: Features used, navigation paths, and interaction patterns
  • Performance Data: App crashes, errors, loading times, and technical issues
  • Device Information: Operating system version, app version, and general device characteristics

1.2 Multiplayer Functionality Data

For multiplayer games, we collect minimal data necessary for functionality:

  • Anonymous Device ID: Anonymous device identifiers for app functionality and multiplayer connections
  • Connection Data: IP address hash (for rate limiting and security), user agent string
  • Game Room Data: Temporary game session information for multiplayer matches

1.3 User Alias Data (Optional)

For leaderboard functionality, you may optionally provide:

  • User Alias: A display name for leaderboard rankings
  • Responsibility: You are responsible for not providing any personalized information in your alias
  • Leaderboard Data: Game scores and rankings associated with your alias

1.4 Game Progress Data

For challenge mode functionality, we store game progress data:

  • Completed Levels: List of challenge levels you have completed (stored in Firestore for cross-device sync)
  • Highest Unlocked Level: The highest challenge level you have unlocked (stored in Firestore)
  • User Flags: Preference flags and dialog states (e.g., which features have been introduced, welcome dialog shown) stored in Firestore

1.5 Local Storage Data

We store certain data locally on your device:

  • Settings: Theme preferences, sound settings, vibration preferences
  • Consent Status: Your analytics consent decision

1.6 Bug Reports (Optional)

If you choose to report bugs, we may collect:

  • Error descriptions and technical details
  • Device information relevant to the issue
  • App state information at the time of the error

2. What We Do NOT Collect

We do not collect the following types of information:

  • Personal identification information (name, email, phone number)
  • Payment information or financial data
  • Location data or GPS coordinates
  • Photos, videos, or other media files
  • Contact lists or address book information
  • Social media account information

3. How We Use Your Information

3.1 Analytics Data

We use analytics data to:

  • Improve game performance and fix bugs
  • Understand which features are most popular
  • Optimize user interface and gameplay experience
  • Make informed decisions about future updates

3.2 Multiplayer Data

We use multiplayer data to:

  • Enable peer-to-peer connections between players
  • Prevent abuse and ensure fair gameplay
  • Maintain game session stability

3.3 User Alias Data

We use user alias data to:

  • Display leaderboard rankings and scores
  • Provide competitive gameplay features
  • Maintain game statistics and achievements

3.4 Game Progress Data

We use game progress data to:

  • Track your challenge level completion and unlock progression
  • Sync your progress across devices
  • Remember which features and dialogs you have seen
  • Provide personalized game experience

3.5 Local Data

We use locally stored data to:

  • Remember your settings and preferences
  • Respect your privacy preferences

4. Data Storage and Security

4.1 Analytics Data

  • Stored securely using Google Firebase Analytics
  • Protected by Google's security measures
  • Automatically deleted after 26 months
  • Never shared with third parties beyond Google Analytics

4.2 Multiplayer Data

  • Stored temporarily in Google Cloud Firestore
  • Automatically deleted when game sessions end
  • Protected by Firebase security rules

4.3 User Alias Data

  • Stored in Google Cloud Firestore for leaderboard functionality
  • Can be deleted by the user at any time
  • Protected by Firebase security rules
  • No personal information is stored with the alias

4.4 Game Progress Data

  • Stored in Google Cloud Firestore for cross-device synchronization
  • Completed levels and highest unlocked level are synced across your devices
  • User flags track preferences and feature introductions
  • Can be deleted by the user at any time
  • Protected by Firebase security rules

4.5 Local Data

  • Stored locally on your device
  • Not transmitted to our servers
  • Remains under your control

5. Your Rights and Choices

5.1 Analytics Consent

  • You can opt-out of analytics at any time in the app settings
  • Disabling analytics does not affect core app functionality
  • Your choice is saved locally and respected across app updates
  • You can change your consent decision at any time

5.2 Data Access and Control

  • You can clear local data by uninstalling the app
  • You can reset game progress through the app settings
  • You can delete your user alias and associated leaderboard data at any time
  • You can delete your game progress data (completed levels, flags) stored in Firestore
  • You can request deletion of any personal data we may have collected

6. Third-Party Services

6.1 Google Services

We use the following Google services:

  • Firebase Analytics: For anonymous usage analytics (with consent)
  • Cloud Firestore: For multiplayer game functionality, game progress synchronization, and user preferences
  • Firebase Auth: For anonymous authentication in multiplayer and challenge modes

6.2 WebRTC

For multiplayer games, we use WebRTC for peer-to-peer communication. This technology:

  • Enables direct connections between players
  • Does not route data through our servers
  • Uses industry-standard encryption

7. Children's Privacy

Our App is intended for users 13 years of age and older. We:

  • Do not knowingly collect personal information from children under 13
  • Do not require personal information for app functionality
  • Use only anonymous data collection methods
  • Comply with applicable children's privacy laws
  • If you are under 13, please do not use this app or provide any information

8. International Data Transfers

Your data may be processed in countries other than your own. We ensure that:

  • All data transfers comply with applicable laws
  • Google's data processing agreements protect your information
  • No personal data is transferred (only anonymous analytics data)

9. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. When we do:

  • We will update the "Last Updated" date at the top of this policy
  • We will notify users through the app or app store updates
  • Continued use of the app constitutes acceptance of the updated policy

10. Contact Information

For privacy concerns or questions about this policy:

  • Contact us through the app store listing
  • Email:
  • Submit bug reports through the app's reporting feature

11. Legal Basis for Processing (GDPR)

For users in the European Union, our legal basis for processing your data is:

  • Consent: For analytics data collection (you can withdraw at any time)
  • Legitimate Interest: For app functionality and security
  • Contract Performance: For multiplayer game functionality
